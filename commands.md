# Essential Git Commands

1. **git init** – Initializes a new Git repository in your current directory.  
2. **git add** – Stages changes (files) to be committed.  
3. **git commit** – Records the staged changes with a descriptive message.  
4. **git status** – Displays the current state of the working directory and staging area.  
5. **git push** – Uploads local commits to a remote repository (e.g., GitHub).
6. **git pull** – Fetches and integrates changes from a remote repository.
7. **git clone** – Copies an existing Git repository into a new local directory.
